![目录结构][1]

### 1、bin目录
主要存放软件启动脚本包括Windows和Linux下启动。
![bin目录][2]

### 2、conf目录
mail.properties
![conf目录][3]
可以通过改变mail.properties，来改变软件的一些参数。
### 3、lib目录
sendmail的核心包和依赖包
![lib目录][4]

### 4、logs目录
软件运行的日志存放目录
![logs目录][5]

### 5、temp目录
程序生成的图片存放的临时目录
![temp目录][6]

### 6、template目录

## 三、Linux下安装
### 1、解压mailsend-0.0.1.tar.gz
![目录结构][20]

![解压][21]
为了方便启动和维护可以添加环境变量
![环境变量][22]

![环境变量][23]

添加可执行权限
```
chmod +x mailsend-0.0.1/bin/sendmail.sh 
```

### 2、启动软件
启动并查看日志：
```  
./sendmail.sh start && ./sendmail.sh log
```
